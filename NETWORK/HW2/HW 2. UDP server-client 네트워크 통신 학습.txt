UDPServer.cpp와 UDPClient.cpp를 그대로 이용하여 서버-클라이언트 통신을 실행하고 결과 출력 (캡쳐하여 보고서에 작성)
- 클라이언트가 5개의 메시지를 보내고 서버가 5개의 메시지를 수신

- 클라이언트가 보낼 5개의 메시지는 다음으로 할 것

⦁ Hello

⦁ Hi

⦁ How are you?

⦁ I am a boy. You are a girl. This is my pen.

⦁ I will do my best for computer network subjects.

- 클라이언트 및 서버에 출력된 것을 출력하여 캡쳐

 

UDP를 이용하여 파일 송수신 후에 파일 내용 출력
- 클라이언트가 “novel.txt”를 서버에게 요청 (novel.txt문서 미리 보기)

>> request “novel.txt” (클라이언트 화면에 다음을 입력)

- 서버는 클라이언트로부터 요청을 받고 “novel.txt”를 클라이언트에게 전송

>> The server received a request from a client (요청 수신 확인되면 서버 화면에 출력)

>> The server sent “novel.txt” to the client (송신 완료하면 서버 화면에 출력)

- 클라이언트는 수신한 “novel.txt”를 저장하고, 화면에 내용 출력

>> The client received “novel.txt” from the server. (“novel.txt”를 수신 완료하면 클라이언트 화면에 출력)

>> (“novel.txt”를 화면에 출력)
